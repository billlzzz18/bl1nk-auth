## 📋 TODO.md Rules

```
สัญลักษณ์:
[ ]  = ยังไม่ได้ทำ
[~]  = กำลังทำ (current session)
[×]  = ทำเสร็จแล้ว
[←]  = ข้ามไปก่อน / รอก่อน (user สั่ง)

กฎ:
- ห้ามลบ item ที่มีอยู่ ใช้ append เท่านั้น

- อัปเดต status ทันทีเมื่อเริ่ม/เสร็จ task
- ถ้า user redirect งาน → mark [←] แล้ว append งานใหม่
```
